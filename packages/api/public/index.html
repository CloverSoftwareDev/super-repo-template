Get Started

Quick Start
Installation

Clone the Superepo and install dependencies. We use `npm` workspaces to manage the monorepo.

git clone https://github.com/CloverSoftwareDev/super-repo-template
npm install
Environment Setup

Each package has its own `.env` requirements. Use the CLI to scaffold them instantly.

npm run cli -- setup-env
Unified Dev Server

Launch all services (API, Web, Website) in parallel with a single command.

npm run dev
packages/cli (Tools)
Overview

Custom internal tooling for scaffolding components and managing the monorepo. Instead of memorizing 20 different npm scripts, use the CLI.

Usage
npm run cli -- --help
packages/api (Fastify + Supabase)
Fastify Supabase Zod
Overview

A High-performance REST API. We use Fastify for its low overhead and Swagger for auto-generated documentation. Supabase provides the Postgres database and Auth layer.

Database Connection

Connection strings are managed via `packages/api/.env`. Use the `supabase` CLI for migrations.

API Documentation

Swagger UI is automatically served at `/documentation`.

npm run dev -w packages/api
curl http://localhost:8080/documentation
packages/api-client (Kubb)
Kubb TanStack Query Faker
Overview

Auto-generated API client, types, and mocks from your OpenAPI specification. Kubb generates type-safe hooks for data fetching.

Regenerate Client
npm run generate -w packages/api-client
packages/client (Expo)
Expo SDK 50 React Native
Overview

Universal mobile application for iOS and Android. It shares the same API types as the web app, ensuring that if the backend changes, the mobile app build fails safely before runtime.

Development
npm run ios -w packages/client
npm run android -w packages/client
packages/web (Next.js 16)
Next.js 16 Vanilla CSS React 19
Overview

The fullstack dashboard application. It serves as the primary interface for users. We explicitly use Vanilla CSS to avoid Tailwind bloat and maintain a cleaner separation of styles.

Directory Structure
src/
├── app/ # App Router pages
├── components/ # Reusable UI components
├── styles/ # Global & module CSS
├── lib/ # Shared utilities
└── GEMINI.md # AI Context
Key Commands
npm run dev -w packages/web # Dev Server (Port 3000)
npm run build -w packages/web # Production Build
packages/website (HTMX)
Overview

Static landing page built with HTMX. Lightweight and fast for marketing and documentation purposes.

Development
npm run dev -w packages/website
