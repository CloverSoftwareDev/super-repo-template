# Client Package

This package contains the Expo/React Native application.

## Tech Stack

- **Expo**: React Native framework
- **React Native Paper**: Material Design 3 UI Library
- **TypeScript**: Static typing

## Best Practices

### UI/UX
- Use `react-native-paper` components for all UI elements to ensure Material Design consistency.
- Use the `theme` prop in `PaperProvider` to customize colors if needed.

### Navigation
- Use React Navigation (install if needed) or Expo Router.

### Platform Specifics
- Check for platform compatibility when using native modules (Web vs Mobile).
