Get Started

Installation & Setup

Bootstrap your new project in seconds.

git clone https://github.com/super-repo/template
npm install
Development Workflow

Start the unified development server. This launches API, Web, and Website packages in parallel.

npm run dev

Access points:

  • Web: localhost:3001
  • Docs: localhost:3000
  • API: localhost:8080
Project Structure

The monorepo is organized into specialized workspaces:


packages/api: Fastify backend with type-safe schemas.
packages/web: Next.js App Router application.
packages/client: Expo React Native application.
packages/cli: Custom scaffolding tools.
AI Context Files

To enable AI collaboration, keep GEMINI.md files updated in each package root. These files serve as the "long-term memory" for Antigravity agents.
