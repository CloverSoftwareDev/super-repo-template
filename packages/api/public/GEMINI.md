# Website Package

This package contains the HTMX landing page.

## Tech Stack

- **HTML5**: Semantic markup
- **HTMX**: For dynamic interactions without writing JS
- **Vanilla CSS**: Material Design 3 tokens
- **Material Web**: Web Components for Material Design elements

## Best Practices

- Use semantic HTML tags (`main`, `article`, `section`)
- Use CSS variables for theming
- Use HTMX for server interactions
- Ensure SEO meta tags are present

### Component Scaffolding
- **ALWAYS** use CLI: `npm run cli -- scaffold component website MyComponent`
- Do NOT manually create component files.
