# Web Package

This package contains the Next.js fullstack application.

## Tech Stack

- **Next.js**: React framework (App Router)
- **Vanilla CSS**: Global styles and CSS modules
- **Material Design 3**: Integration via CSS variables (Tokens)

## Best Practices

### CSS
- Use variables defined in `src/styles/globals.css` for colors and typography.
- Avoid utility classes (Tailwind) unless explicitly added.
- Use Semantic HTML.

### API
- Use `@super-repo/api-client` to fetch data from the API.
